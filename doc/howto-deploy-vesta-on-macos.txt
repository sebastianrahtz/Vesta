1) run ant
2) create Vesta.app with jar bundler(Developer/Application/Utilities) (do not build jar in eclipse dir)
3) change plist file and add
       <key>StartOnMainThread</key>
       <true/>
   to the java section
4) cp -r ../Stylesheets2/ Vesta.app/Contents/Resources/resources/stylesheets
5) run 'hdiutil create -srcfolder Vesta.app vesta.dmg' to create dmg
6) copy dmg to binaries
